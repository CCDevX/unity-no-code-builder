Centre d'aide

Bienvenue sur la page d'aide

Cette section vous guidera à travers les principales fonctionnalités de l'application et vous aidera à démarrer avec la création de plugins Unity.

  • Créer un projet Cliquez sur "Créer un projet" dans le tableau de bord pour commencer un nouveau plugin Unity. Cela ouvrira l'assistant de projet où vous pourrez configurer les paramètres de base et la structure de votre plugin.
  • Projet d'exemple Essayez l'exemple préchargé pour explorer l'interface. Cette approche pratique vous aidera à comprendre comment les différents composants fonctionnent ensemble.
  • Constructeur de projet Utilisez l'éditeur visuel pour ajouter et configurer des composants pour votre plugin. Glissez-déposez des éléments, définissez des propriétés et prévisualisez votre plugin en temps réel.
  • Exporter et déployer Une fois prêt, exportez votre package de plugin pour l'utiliser dans Unity. Le système générera tous les fichiers nécessaires et fournira les instructions d'installation.
